# Firebase Authentication

Implementing FirebaseAuth by Email and Password, and GoogleSignIn.

## For developers
> To clone locally
```
$ git clone 
$ cd 
```
> Open the project in an editor, if you use VS Code, follow this
```
$ cd 
$ code .
```

> To run, follow this on terminal
```
$ flutter clean
$ flutter pub get
$ flutter run --release
```
